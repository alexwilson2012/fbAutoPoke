# Facebook AutoPoke

##### By Alex Wilson


#### Installation / Setup
1. Install PhantomJS and CasperJS
2. Setup a cronjob that runs the following command

casperjs <path-to-/fbCasperPoke.js> <Facebook-ID-of-person-to-poke> <Facebook-login-email> <Facebook-login-password>


#### Requirements
PhantomJS  
CasperJS  
Crontab
